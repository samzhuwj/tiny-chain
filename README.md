# tinychain
A mini blockchain implementation in C++14

## toolchain
* gcc/clang support C++14
* boost 1.56+ (datetime)
* cmake
* git

## build
On workpath of tinychain:
```
$ mkdir build
$ cmake ..
$ make
```

## run
On workpath of tinychain:
```
$ cd build/bin
$ mkdir html && cp ../../etc/index.html html
$ ./tinychain
```
